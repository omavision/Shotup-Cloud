# M1 — Shotup Cloud Foundation

## Goal

Establish the production foundation for Shotup Cloud.

## Scope

- Vapor API foundation
- Feature-based architecture
- PostgreSQL + Fluent
- Environment configuration
- Health endpoint
- API versioning
- Initial user model
- Initial authentication planning
- GitHub repository setup

## Out of Scope

- iOS app sync
- Media upload
- Cloudflare R2
- Web dashboard
- Crew collaboration
- AI features
- Payments

## Success Criteria

- API builds successfully
- Server runs locally
- `/health` endpoint responds
- PostgreSQL configuration is ready
- First migration structure exists
- Repository has clean Git history
